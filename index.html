📱 How to Use ×

🎯 STEP 1: Weight Select Karo

• Top par "WEIGHT" section mein ◀ ▶ arrows par tap karo

• Apna weight select karo (300g, 500g, 750g, 1kg, etc.)

• Weight change hone par formula automatically update hoga

🔢 STEP 2: Calculation Karo

📦 Current Shipper:

→ Shipper count daalo, automatically Pouch aur Pallet calculate hoga

🚛 Pallet Count:

→ Pallet count daalo, automatically Shipper aur Pouch calculate hoga

📦 Current Pouch:

→ Pouch count daalo, automatically Shipper aur Pallet calculate hoga

📊 STEP 3: Results Dekho

Neeche "Results" section mein dikhega:

  • 📦 Total Pouches
  • 📦 Total Shippers
  • 🚛 Total Pallets

📋 STEP 4: Formula Dekho

Results ke neeche formula dikhega:

  • ✓ Pouch = Shipper
  • ✓ Shipper = Pallet
  • ✓ Pouch = Pallet
  • ⚡ Speed: Pouch/min

⚙️ STEP 5: Custom Formula (Optional)

1. Settings (⚙️) mein jao

2. "Manual Formula" toggle ON karo

3. Form fill karo aur Save karo

4. Saved formulas list mein dikhenge

💡 Tips

  • ✓ Sirf ek field mein number daalo
  • ✓ Baaki auto calculate hoga
  • ✓ Refresh button (↻) se reset karo
  • ✓ Dark mode (☀️/🌙) switch kar sakte ho
